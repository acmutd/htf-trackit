Intro to Computer Graphics

Speakers: George Goodman and Matthew Vu
Description of workshop:
Come on and ZOOM into some cool graphics as you learn how to use textures to apply an image onto a 3D surface!
Please have these downloaded and available:

  • An IDE of your choice (Visual Studio Code recommended)
  • Ensure WebGL is supported by your browser
  • A local HTTP server (see the README in our workshop code repo)

Link to code: https://github.com/acmutd/htf-community
