How to open a pull request:

  1. On GitHub, navigate to the main page of the repository.

  2. In the “Branch” menu, choose the branch that contains your commits.

  3. Above the list of files, click on Pull request.

  4. Use the base branch dropdown menu to select the branch you’d like to merge your changes into, then use the compare branch drop-down menu to choose the topic branch you made your changes in.

  5. Type a title and description for your pull request.

  6. To create a pull request that is ready for review, click Create Pull Request. To create a draft pull request, use the drop-down and select Create Draft Pull Request, then click Draft Pull Request.

After your pull request has been reviewed, it can be merged into the repository.

How to check if a pull request was submitted using the Hacktoberfest website:

  • Log in with your GitHub account at hacktoberfest.digitalocean.com to check your progress and stats.
  • There is a pull request counter at the top of the Profiles page.
  • There is also a legend at the bottom of the page deciphering the meaning behind various symbols that will appear in the “Your Contributions” tab in your profile. Check there to track the progress of your pull request!
